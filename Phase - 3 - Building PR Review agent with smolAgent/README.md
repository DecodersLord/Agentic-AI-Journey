### Why a separate repo?
As the agentic AI journey progresses, fully-featured apps (like the PR Review Agent with multi-tool dispatch, API integration, and framework usage) are better suited to have their own release lifecycle, issue tracking, and clean commit history. This `Agentic AI Journey` repository acts as the core hub tracing the learning journey.
